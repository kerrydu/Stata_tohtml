Stata tohtml Examples

This page provides three examples showing how to use tohtml to convert Stata logs, commands, results, figures, and tables into HTML documents. The three examples are based on the same Stata workflow and contain the same figures and tables, but they use different output modes of tohtml.

Example 1: detail mode

Detailed HTML Output

This example demonstrates the detail mode of tohtml. It keeps a relatively complete record of the Stata workflow, including commands, selected log contents, figures, and regression output. This mode is useful when users want a more comprehensive HTML record of their analysis.

Example 2: clean mode

Clean HTML Output

This example demonstrates the clean mode of tohtml. It produces a cleaner HTML document by reducing unnecessary log information while retaining the main analytical contents, including figures and tables. This mode is suitable for users who prefer a more readable report-style output.

Example 3: cleancode mode

Cleancode HTML Output

This example demonstrates the cleancode mode of tohtml. It focuses on presenting clean Stata code together with the corresponding outputs, figures, and tables. This mode is useful when users want to share reproducible examples with concise and readable code.
